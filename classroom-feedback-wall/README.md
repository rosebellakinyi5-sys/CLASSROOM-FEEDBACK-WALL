# Classroom Feedback Wall

## Learning goals

This beginner project practises:

- Selecting and updating DOM elements
- Creating feedback cards with JavaScript
- Handling `submit` and `click` events
- Reading textareas, selects and checkboxes
- Conditional logic
- Adding CSS classes based on form data
- Filtering displayed content
- Writing simple DOM tests

## Student task

1. Open `index.html`.
2. Create the required form inside `#form-workspace`.
3. Include labels for every form control.
4. Include at least three buttons: Post Feedback, Show Positive and Clear Feedback.
5. Complete the functionality in `script.js`.
6. Use the prompts in `tests.js` to test the DOM.

## Required element IDs

- `feedback-form`
- `student-name`
- `feedback-message`
- `mood`
- `anonymous`
- `form-message`
- `show-positive`
- `clear-feedback`

## Success criteria

- The page does not refresh when the form is submitted.
- Empty feedback messages are rejected with a visible message.
- A valid submission creates a feedback card.
- Anonymous submissions show “Anonymous”.
- Each mood has a different card style.
- The positive filter hides other moods.
- The clear button removes all feedback.
- The feedback count and current view are updated.
